Bitrix Tips

Stumbled upon the following url when attempting to fetch the latest deals that have been modified: 

your-bitrix-url/crm.deal.list?order[DATE_MODIFY]=DESC

Should get a collection of deals that are in order starting with most recently modified. This way we can get an up to date view of which deals have more activity and which are just "existing".

Use a do..while loop (check to see if there is a next property in the response body) to repeat the GET request until all deals are received.
